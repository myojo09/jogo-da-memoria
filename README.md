# 🐶 Jogo da Memória - Raças de Cachorros

Um jogo da memória interativo desenvolvido com **HTML, CSS e JavaScript**, utilizando imagens de diferentes raças de cachorros.

## 🎮 Sobre o Projeto

O objetivo do jogo é encontrar todos os pares de cartas no menor tempo possível e com a menor quantidade de movimentos.

O projeto foi desenvolvido com foco em:

- Manipulação do DOM
- Lógica de jogos em JavaScript
- Responsividade
- Animações CSS
- Experiência do usuário (UX)

## ✨ Funcionalidades

- Tela inicial personalizada
- Cartas embaralhadas automaticamente
- Contador de movimentos
- Cronômetro em tempo real
- Sistema de pontuação
- Tela de vitória
- Botão para reiniciar a partida
- Layout responsivo para desktop e mobile
- Animações de virar cartas
- Imagens reais de raças de cachorros

## 🐕 Raças Presentes

- Golden Retriever
- Labrador Retriever
- Pastor Alemão
- Husky Siberiano
- Poodle
- Shih Tzu
- Beagle
- Border Collie
